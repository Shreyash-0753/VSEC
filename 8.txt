1. Open new folder
 - Create app.py
 - Create requirements.txt(Flask)
 - Create Dockerfile

2. On Terminal :
    docker --version
    minikube version
    minikube start --driver = docker
    kubectl get nodes
    eval $(minikube docker-env)
    docker build -t folder_name .
    minikube images | grep folder_name
    minikube image load folder_name
    minikube dashboard

3. On Minikube Dashboard
  - Add yaml file from '+' at top right corner and upload
  - Check pods
  - Add service yaml file similarly from '+' and upload

4. Open new terminal 
      minikube service ai-app-service

5. Again open new terminal
      minikube addons enable metrics-server
      kubectl top pods

6. Open pods on dasboard, wait for 2 mins and reload the page

7. Open new terminal 
      kubectl autoscale deployment ai-app --cpu-percent=50 --min=2 --max=10
8. app.py :
from flask import Flask, request, jsonify
app = Flask(__name__)

@app.route("/")
def home():
 return "AI Model Running in Kubernetes"

@app.route("/predict", methods=["POST"])
def predict():
 data = request.json["number"]
 result = data * 2
 return jsonify({"result": result})

if __name__ == "__main__":
 app.run(host="0.0.0.0", port=5000)

9. dockerfile : 
FROM python:3.9
WORKDIR /app
COPY . .
RUN pip install flask
EXPOSE 5000
CMD ["python","app.py"]
